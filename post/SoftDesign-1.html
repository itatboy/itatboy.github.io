嵌入式软件开发编码规范

嵌入式 编码规范
软件设计
发布日期:   2021-12-01

规范目的

一、规范目的

描述嵌入式软件开发流程中，代码编写及命名规则，有利于项目开发和维护，增强可读性、规范性、复用性。

二、变量命名

变量类型 前缀 举例 重定义类型
char c char cLength; int8_t
short s short sLength; int16_t
int i int iLength; int32_t
long l long lLength; int64_t
unsigned char uc unsigned char ucLength; uint8_t
unsigned short us unsigned short usLength; uint16_t
unsigned int ui unsigned int uiLength; uint32_t
unsigned long ul unsigned long ulLength; uint64_t
bool b 或 is bool bLength;
float f float fLength;
double float d double float dLength;
枚举 e enum eLENGTH;
结构体 str INFO strLength;
指针 p char* pLength;
结构体指针 pstr INFO* pstrLength;
结构体数组 astr INFO astrLength[];
数组 a char aLength[];

全局变量前加 g，例如全局结构体：gSysInfo;
结构体示例

/* 缓存队列数据结构 */
typedef struct _UARTQUEUE
{
    uint16_t usHead;
    uint16_t usTail;
    uint16_t usLenght;
    uint16_t ucBuff[UARTBUFF_LEN * 2 + 2];
} UARTQUEUE;

四、排版

缩进

缩进使用制表符，一个制表符等于4个空格。

五、注释

单行注释

在语句末尾添加，空格后输入单行注释符 //，注释符号空格后输入注释文。

sysDelayms(1000); // 延时1000毫秒

多行注释

在功能块前添加，使用多行注释符/**/，注释文前后各空一格。

/* 满足条件时清空值 */
if(ucFlag)
{
    ucValue = 0;
}

函数注释

在函数前添加，示例如下

/**
  * @brief 发送数据
  * @param ucComID:端口号
  * @param *pData:数据
  * @param ucLength:长度
  * @return 错误代码
  */
int sysComSend(uint8_t ucComID, uint8_t *pData, uint16_t usLength)
